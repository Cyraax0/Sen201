# Student Result Management System

## Project Description
The Student Result Management System is a simple Python-based application
designed to collect a student's name and score, calculate the corresponding
grade, and display the result.

This project was developed as part of the SEN 201 course assignment and
demonstrates the use of variables, functions, and basic input/output in Python.

---

## Features
- Accepts student name and score
- Automatically calculates grade
- Displays student result clearly
- Simple and easy to use

---

## Technologies Used
- Python 3

---

## How the System Works
1. The user enters the student's name.
2. The user enters the student's score.
3. The system calculates the grade using a function.
4. The result is displayed on the screen.

---

## Grade Calculation Criteria
- 70 and above → A  
- 60 – 69 → B  
- 50 – 59 → C  
- 45 – 49 → D  
- Below 45 → F
